# 1161. Maximum Level Sum of a Binary Tree

- **Core Pattern:** BFS
- **Tricky Edge Case:** Negative values

## The "Aha!" Moment

BFS algorithm, level by level

## Complexity

- **Time:** O(N)
- **Space:** O(width)

## Alternative Approaches

DFS with external state tracking the level sums, and pass the current node level state with parameter
